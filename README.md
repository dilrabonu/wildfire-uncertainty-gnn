# Wildfire Uncertainty GNN

Uncertainty-Calibrated Intervention-Aware Spatiotemporal GNN for Wildfire Spread on Stochastic Simulation Data.

## Project Overview
This project studies wildfire spread prediction using graph neural networks with calibrated uncertainty estimation and intervention-aware analysis.

## Objectives
- Model wildfire spread over spatial landscapes
- Quantify predictive uncertainty
- Evaluate calibration quality
- Simulate intervention effects on spread dynamics
